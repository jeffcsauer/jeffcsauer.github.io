Welcome to the new and improved (i.e. streamlined) site! After many hours of chasing what I thought thought was a good website, I realized that simplicity and scalability was key. This site will act as a one-stop-shop for a brief biosketch, blog posts, class materials, a regularly updated CV, and noteworthy news.

In pursuit of efficiency, I switched from the Hugo Academic theme to the Hugo Xmin theme written by the prodigious Yihui Xie. At the near smallest, Yihui states that this theme ‘This XMin theme contains about 130 lines of code in total, including the code in HTML templates and CSS (also counting empty lines).’
